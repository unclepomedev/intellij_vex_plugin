# intellij_vex_plugin

<!-- Plugin description -->
Houdini VEX assyst plugin for JetBrains IDEs (IntelliJ IDEA, PyCharm, WebStorm, etc.)

## Features

* Code Formatting
* Autocomplete
* Error and Warnings (Syntax, Type, etc.)
* Quick Documentation
* Code Navigation
* Inlay Hints
* Rename
* Create a new .vex file

source code: https://github.com/unclepomedev/intellij_vex_plugin
