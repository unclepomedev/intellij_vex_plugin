"""Returns an array of point transforms from an array of point indices."""

:usage: `matrix[] pointtransforms(<geometry>, int pnts[])`
:usage: `matrix[] pointtransforms(<geometry>)`

Returns an array of transforms associated with the point indices, using the [standard instancing point attributes|/copy/instanceattrs].
Returns all the point transforms if the point indices argument is omitted.

:include _common#geometry:

:arg:pnts:
    The array of point indices to query.
