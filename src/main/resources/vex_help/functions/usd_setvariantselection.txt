"""Sets the selected variant in the given variant set."""

:usage: `int usd_setvariantselection(<stage>, string primpath, string variantset, string variant)`

This function sets the selected variant in a given variant set.

:include _common#usd_stagehandle:

:include _common#usd_primpath:

:arg:variantset:
    The variant set name.

:arg:variant:
    The name of a variant to select.

:returns:
    The value of `stagehandle` on success or `-1` on failure.

@examples 
{{{
#!vex
// Set the variant "cone" in a variant set "shapes" on the "shape_shifter" primitive.
usd_setvariantselection(0, "/geo/shape_shifter", "shapes", "cone");
}}}
