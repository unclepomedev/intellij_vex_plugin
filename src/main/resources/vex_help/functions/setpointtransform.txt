"""Sets the world space transform of a given point"""

:usage: `int setpointtransform(int geohandle, int pt, matrix transform)`
:usage: `int setpointtransform(int geohandle, int pt, matrix transform, int constrain)`

Sets the `v@P` and the `3@transform` attributes on the given point from the given 4x4 matrix.

:include _common#geohandle_arg:

:arg:pt:
    The point index to modify.

:arg:transform:
    The 4x4 transform.

:arg:constrain:
    When True, update the children point transforms when modifying a point world 
    transform. When False, the children points stay in place like when using 
    Child Compensation on a transform handle. Defaults to True;
