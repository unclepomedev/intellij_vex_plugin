"""Returns the length (in seconds) of an agent's animation clip."""

:usage: `float agentcliplength(<geometry>, int prim, string clipname)`
:usage: `float agentcliplength(<geometry>, int prim, int clipindex)`

Returns `0` if `prim` is out of range or is not an agent primitive, or if `clipname` is not one of the agent's [animation clips|Vex:agentclipcatalog].

:include _common#geometry:

:arg:prim:
    The primitive number.

:arg:clipname:
    The name of the animation clip.
