"""Remove a target from the primitive's relationship"""

:usage: `int usd_removerelationshiptarget(int stagehandle; string primpath; string name; string target)`

This function removes a target from the primitive's relationship.

:include _common#usd_stagehandle:

:include _common#usd_primpath:

:arg:name:
    The relationship name.

:arg:target:
    The the target path to remove.

:returns:
    The value of `stagehandle` on success or `-1` on failure.

@examples 
{{{
#!vex
// Remove the sphere from cube's relationship.
usd_removerelationshiptarget(0, "/geo/cube", "relationship_name", "/geo/sphere");
}}}
