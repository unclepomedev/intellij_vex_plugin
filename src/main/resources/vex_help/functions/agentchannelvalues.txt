"""Returns the current values of an agent primitive's channels."""

:usage: `float[] agentchannelvalues(<geometry>, int prim)`

If only a single channel value is needed, using [Vex:agentchannelvalue] instead can be significantly faster.

Returns an empty array if `prim` is out of range or is not an agent primitive.

:include _common#geometry:

:arg:prim:
    The primitive number.
