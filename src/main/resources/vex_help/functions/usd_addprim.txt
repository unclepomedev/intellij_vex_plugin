#type: vex

#group: usd
#since: 18.0

"""Creates a primitive of a given type."""

:usage: `int usd_addprim(int stagehandle, string primpath, string typename)`

This function creates a new primitive of a given type and at a given path, if it one does not exist yet.

:include _common#usd_stagehandle:

:include _common#usd_primpath:

:arg:typename:
    The name or an alias of the type.

:returns:
    The value of `stagehandle` on success, or `-1` on failure.

@examples 
{{{
#!vex
// Adds a sphere primitive.
usd_addprim(0, "/geo/sphere", "Sphere");
}}}

@related
- [Vex:usd_isprim]
