"""Returns a Zeltner Sheen BSDF."""

:usage: `bsdf sheen(vector nml, float roughness, ...)`

    Returns Zeltner Sheen BSDF.
    
    See [writing a PBR shader|/vex/pbr] for information on BSDFs.

:arg:nml:
    The normal of the surface to use for evaluate.

:arg:roughness:
    Low value can produce high-sheen fabric like appearance, while high value
    can produce dusty appearance.
