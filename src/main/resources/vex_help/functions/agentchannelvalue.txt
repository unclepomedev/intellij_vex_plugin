"""Returns the current value of an agent primitive's channel."""

:usage: `float agentchannelvalue(<geometry>, int prim, int channel)`

Returns zero if `prim` is out of range, `prim` is not an agent primitive, or `channel` is out of range.

:include _common#geometry:

:arg:prim:
    The primitive number.

:arg:channel:
    Index of a channel in the agent's rig.
