#type: vex
#context: all
#tags: usd, usd_prim

#since: 17.5

"""Returns the primitive's purpose."""

:usage: `string usd_purpose(<stage>, string primpath)`

This function returns the given primitive's purpose, e.g., "default", "render", "proxy", "guide", etc.

:include _common#usd_stage:

:include _common#usd_primpath:

:returns:
    The purpose of the given primitive.

@examples 
{{{
#!vex
// Get the sphere primitive's purpose.
string purpose = usd_purpose(0, "/geo/sphere");
}}}

@related
- [Vex:usd_setpurpose]
