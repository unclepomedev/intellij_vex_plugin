= usd_relbbox =

#type: vex
#context: all
#tags: usd, usd_bbox
#group: usd
#since: 18.0

"""Returns the relative position of the point given with respect to the bounding box of the geometry."""

:usage: `vector usd_relbbox(<stage>, string primpath, string purpose, vector position)`

Returns the relative position of the given point with respect to the bounding box of the primitive.


:include _common#usd_stage:

:include _common#usd_primpath:

:arg:purpose:
    The primitive's purpose for which to use the bounding box (e.g., "render").

:returns:
    The relative position of the given point with respect to the bounding box of the primitive.

@examples 
{{{
#!vex
// Get the points relative position.
vector pt = {1, 0, 0};
vector rel_pt = usd_relbbox(0, "/src/sphere", "render", pt);
}}}
