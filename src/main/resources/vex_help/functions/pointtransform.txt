"""Returns a point transform from a point index."""

:usage: `matrix pointtransform(<geometry>, int pnt)`

Computes a transform for the point, using the [standard instancing point attributes|/copy/instanceattrs].

:include _common#geometry:

:arg:pnt:
    The point index to query.
