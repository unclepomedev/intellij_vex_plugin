"""Sets the targets in the primitive's relationship"""

:usage: `int usd_setrelationshiptargets(int stagehandle; string primpath; string name; string[] targets)`

This function sets the targets in the primitive's relationship.

:include _common#usd_stagehandle:

:include _common#usd_primpath:

:arg:name:
    The relationship name.

:arg:targets:
    The the target paths to set the relationship to.

:returns:
    The value of `stagehandle` on success or `-1` on failure.

@examples 
{{{
#!vex
// Set the cube's relationship.
usd_setrelationshiptargets(0, "/geo/cube", "new_relation", array("/geo/sphere6", "/geo/sphere7"));
}}}
