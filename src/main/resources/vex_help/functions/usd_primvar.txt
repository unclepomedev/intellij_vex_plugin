= usd_primvar =

#type: vex
#context: all
#tags: usd, usd_primvar
#group: usd
#since: 18.0

"""Reads the value of a primvar directly from the USD primitive."""

:usage: `<type> usd_primvar(<stage>; string primpath; string name)`
:usage: `<type>[] usd_primvar(<stage>; string primpath; string name)`
:usage: `<type> usd_primvar(<stage>; string primpath; string name; float timecode)`
:usage: `<type>[] usd_primvar(<stage>; string primpath; string name; float timecode)`

This function returns a value of a primvar on a given primitive.

:include _common#usd_stage:

:include _common#usd_primpath:

:arg:name:
    Primvar name (without namespace).

:include _common#usd_timecode:

:returns:
    The value of an existing primvar, or zero/empty value if the primvar does not exist. Use [Vex:usd_isprimvar] if you want to check whether the primvar exists.

@examples 
{{{
#!vex
// Get the value of some primvars on the cube primitive.
vector vec_value = usd_primvar(0, "/geo/cube", "vec_primvar_name"); 
float values[] = usd_primvar(0, "/geo/cube", "primvar_name");
float value    = usd_primvar(0, "/geo/cube", "primvar_name", 3);

v[]@foo_at_current_frame = usd_primvar(0, "/geo/sphere", "foo");
v[]@foo_at_frame_8       = usd_primvar(0, "/geo/sphere", "foo", 8.0);
}}}
