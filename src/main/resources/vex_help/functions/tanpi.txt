"""Returns the trigonometric tangent of the argument scaled by Pi"""

:usage: `float tanpi(float n)`

    Returns the trigonometric tangent of `n`, where `n` is in
    semi-circles.

:usage: `vector2 tanpi(vector2 v)`
:usage: `vector tanpi(vector v)`
:usage: `vector4 tanpi(vector4 v)`

    Returns a new vector with `tanpi()` applied to each component.
