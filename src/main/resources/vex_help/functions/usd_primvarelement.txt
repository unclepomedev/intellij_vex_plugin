= usd_primvarelement =

#type: vex
#context: all
#tags: usd, usd_primvar
#group: usd
#since: 18.0

"""Reads the value of an element from the array primvar directly from the USD primitive."""

:usage: `<type> usd_primvarelement(<stage>; string primpath; string name; int index)`
:usage: `<type> usd_primvarelement(<stage>; string primpath; string name; int index; float timecode)`

This function returns a value of an element in given array primvar on a given primitive.

:include _common#usd_stage:

:include _common#usd_primpath:

:arg:name:
    Primvar name (without namespace).

:arg:index:
    The index into the array. 

:include _common#usd_timecode:

:returns:
    The value of an element in an existing array primvar, or zero/empty value if the primvar does not exist. Use [Vex:usd_isprimvar] if you want to check whether the primvar exists.

@examples 
{{{
#!vex
// Get the value of some primvars on the cube primitive.
float value    = usd_primvarelement(0, "/geo/cube", "primvar_name", 3);

v@element_2_at_current_frame  = usd_primvarelement(0, "/geo/sphere", "foo", 2);
v@element_2_at_frame_8        = usd_primvarelement(0, "/geo/sphere", "foo", 2, 8.0);
}}}
