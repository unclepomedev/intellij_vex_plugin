= usd_bindmaterial =

#type: vex
#context: all
#tags: usd, usd_prim, usd_material
#group: usd
#since: 21.0

"""Binds a material to a given primitive."""

:usage: `int usd_bindmaterial(int stagehandle; string primpath; string materialpath)`

This function assigns the given material to the given primitive.

:include _common#usd_stagehandle:

:include _common#usd_primpath:

:arg:materialpath:
    The path of the material primitive to assign.

:returns:
    The value of `stagehandle` on success or `-1` on failure.

@examples 
{{{
#!vex
// Assigns a metal material to the sphere geometry.
usd_bindmaterial(0, "/geo/sphere", "/materials/metal");
}}}
