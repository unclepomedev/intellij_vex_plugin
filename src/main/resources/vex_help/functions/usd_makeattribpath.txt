"""Constructs an attribute path from a primitive path and an attribute name."""

:usage: `string usd_makeattribpath(<stage>; string primpath; string name)`

This function returns the full path of a given attribute.

:include _common#usd_stage:

:include _common#usd_primpath:

:arg:name:
    Attribute name.

:returns:
    The full path of a given attribute.

@examples 
{{{
#!vex
// Obtain the full path to the attribute "attrib_name" on the cube primitive.
string attrib_path = usd_makeattribpath(0, "/geo/cube", "attrib_name");
}}}
