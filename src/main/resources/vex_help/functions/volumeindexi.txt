#type: vex
#context: all

"""Gets the integer value of a specific voxel."""

:usage: `int volumeindexi(<geometry>, int primnum, vector voxel)`
:usage: `int volumeindexi(<geometry>, string volumename, vector voxel)`

:include _common#geometry:

:returns:
    The integer value of a specific voxel in a volume primitive.
    
    Returns 0 if `primnum` or `inputnum` is out of range, the geometry
    is invalid, or the given primitive is not a volume primitive.
