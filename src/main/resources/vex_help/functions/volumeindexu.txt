= volumeindexu =

#type: vex
#context: all
#tags: volume
#group: volume

"""Gets the vector2 value of a specific voxel."""

:usage: `vector2 volumeindexu(<geometry>, int primnum, vector voxel)`
:usage: `vector2 volumeindexu(<geometry>, string volumename, vector voxel)`

:include _common#geometry:

:returns:
    The vector2 value of a specific voxel in a volume primitive.
    
    Returns 0 if `primnum` or `inputnum` is out of range, the geometry
    is invalid, or the given primitive is not a volume primitive.
