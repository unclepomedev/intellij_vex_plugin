"""Checks if the primitive is active."""

:usage: `int usd_isactive(<stage>, string primpath)`

This function checks whether the given primitive is active.

:include _common#usd_stage:

:include _common#usd_primpath:

:returns:
    1 if the primitive is active, and 0 otherwise.

@examples 
{{{
#!vex

int is_active = usd_isactive(0, "/geometry/sphere");
}}}
